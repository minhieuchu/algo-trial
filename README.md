# algo-trial

## Description
Provides an API for basic backtesting strategies.

## Installation
```bash
# Create a virtual environment and activate
pip install -r requirements.txt
```

## Run
```bash
python main.py
```

## Test
```bash
cd tests
pytest -v
```

## Run tests with coverage
```bash
cd tests
coverage run -m pytest -v
```

### View test coverage report
```bash
coverage report -m
```
